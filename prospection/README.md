# Suivi de Prospection IndHack

Ce dossier contient les outils pour suivre tes campagnes de mailing et prospection.

## Fichier de suivi
Le fichier principal est `suivi_prospection.csv`. 

### Comment l'utiliser avec Excel / Google Sheets :
1. Ouvre Excel ou Google Sheets.
2. Fais **Fichier > Importer** (ou Ouvrir) et sélectionne `suivi_prospection.csv`.
3. Assure-toi que le délimiteur est la **virgule** (`,`).
4. Tu peux maintenant trier par catégorie (ex: "Coiffeur Nice") et suivre tes relances.

### Colonnes incluses :
- **Date** : Jour de l'envoi.
- **Catégorie** : Pour segmenter tes envois (ex: Coiffeur Nice, Resto Paris, etc.).
- **Nom de l'établissement** : Le nom pour personnaliser tes mails.
- **Email** : L'adresse de contact.
- **Statut** : À envoyer / Envoyé / Relancé / RDV / Refus / Client.
- **Lien Audit** : Si tu as généré un rapport spécifique sur indhack.com.
- **Notes** : Historique des échanges.
