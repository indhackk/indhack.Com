# IndHack Redesign - Next.js

Ce projet est une refonte moderne du site indhack.com, construite avec **Next.js 14**, **Tailwind CSS** et **Framer Motion**.

## 🚀 Pré-requis

Comme nous avons détecté que Node.js n'était pas installé sur votre machine, vous devez l'installer pour faire fonctionner le site.

1.  Téléchargez et installez **Node.js** (version LTS recommandée) depuis : [nodejs.org](https://nodejs.org/)
2.  Une fois installé, rouvrez votre terminal.

## 🛠️ Installation

Dans le dossier du projet (`/Users/indiana/Desktop/indhack.Com`), lancez la commande suivante pour installer les dépendances :

```bash
npm install
```

## ▶️ Lancer le site

Pour voir le site en local :

```bash
npm run dev
```

Ouvrez ensuite [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## 📁 Structure

*   `/app` : Les pages du site (`home`, `about`, `solutions`, `blog`).
*   `/components` : Les composants réutilisables (Navbar, Hero, Modal, etc.).
*   `/public` : Images et fichiers statiques.

## 🎨 Design

Le design utilise une approche "Premium" avec :
*   Police : **Outfit** (Google Fonts).
*   Couleurs : Palette issue de votre marque.
*   Animations : Framer Motion pour des transitions fluides.
