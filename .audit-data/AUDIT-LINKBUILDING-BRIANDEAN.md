## 4. Data PR / Observatoire annuel GEO France

Le baromètre "Sites français invisibles IA 2026" est un **prototype data PR**, pas encore une campagne. Voici ce qui manque pour le transformer en machine à backlinks récurrente.

### Diagnostic actuel

✅ Existant : article blog 2 500 mots, page hub `/etude/barometre-visibilite-ia-2026`, données chiffrées sourcées
❌ Manquant : communiqué de presse PDF, infographie partageable, embed widget, méthode reproductible documentée, partenariat institutionnel

### Concept "Observatoire GEO France" (récurrent, annuel)

Positionner indhack.com comme l'**institution data** sur la visibilité IA en France. Modèle : Yougov, Médiamétrie, Data.gouv.fr.

**Format livrable annuel** :
- Étude crawlable de 1 000 sites FR (top trafic Similarweb)
- Mesure de blocage robots.txt par crawler IA (8 crawlers, 8 secteurs)
- Citation rate dans ChatGPT/Perplexity/Claude/Gemini sur 50 requêtes de marque
- Rapport PDF 30 pages + infographie + dashboard interactif
- Date fixe annuelle (ex : "Observatoire GEO 2027 publié le 1er avril")

**Pourquoi ça marchera** :
- Pas de concurrent FR sur ce positionnement
- Médias adorent les chiffres "X % des entreprises FR sont invisibles"
- Effet flywheel : chaque édition cite la précédente (auto-backlinking)
- Lead gen : 30+ entreprises voudront leur audit perso

**Backlinks projetés** : 50+/an récurrents (medias FR + comparatifs SEO + LinkedIn shares de DG)

### Production tactique 30 jours

1. Créer `/observatoire-geo-france/` avec compteur "Édition 2027 — J-365"
2. Packager données 2026 en PDF 24 pages design + infographie SVG embarquable
3. Communiqué de presse 1 page format AFP
4. Embed widget : `<iframe>` "Votre site est-il invisible pour ChatGPT ?" — chaque embed = backlink
5. Outreach pré-publication : envoyer données en exclu à BDM 48h avant tout le monde
